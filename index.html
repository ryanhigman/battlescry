🔮 BattleScry — Quick Start✕
Getting Started
1. Run !bscry in Discord during combat
2. Click the BattleScry link in the embed
3. Make your changes — move tokens, edit overlays
4. Copy the command and paste it back in Discord
Map Controls
• Drag tokens to move them on the grid
• Click + Shift to multi-select tokens
• Scroll wheel or use zoom controls to zoom
• Ctrl+Z to undo changes
Coordinate Placement
• Click a coordinate field (Position, Origin, End)
• Cursor changes to crosshair ⊕
• Click anywhere on the map to set the coordinate
• Zoom while in crosshair mode without losing focus
Overlays
• Use the dropdown at top of sidebar to add overlays
• Attach to (-t) makes overlays persistent
• {targ} makes origin follow the attached token
• Aim at makes end follow another token
• Drag overlay handles to resize and reposition
Token Images
• Click Manage Custom to set token art
• Browse 1400+ SRD monster tokens
• Enter shortcodes directly or upload custom images
• Face Detect uses OTFBM's facial recognition for better cropping
Round-Trip
• All changes go through !bscry apply
• Notes update via set_note() — no -t conflicts
• Map auto-refreshes in the Discord embed
• Run !bscry again for a new session
