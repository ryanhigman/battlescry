Welcome to BattleScry!
BattleScry builds D&D battlemaps visually. Upload your own map and token art, drag combatants into place and resize them, and it writes the Avrae commands for you ... no manual image hosting, grid alignment, or command syntax to wrestle with.

Made for Avrae combat on Discord. Already in a game? Run !bscry during combat and click the link to edit your live map.
